# Sharrim planning material

Working sources for the seven sibling-tractate title pages. Two standing
rulings (project owner, PR #9 review):

**TOC files are citation metadata, not reproduced text.** Chapter/section
titles plus page numbers are the same class of information the Ein Mishpat
glosses carry (book + page, so readers can find the real text) and the
same mapping the whole tractate is built on (folio number = source page).
SPEC §0 governs folio *content* — the expressive text of the books, which
these files do not contain. The built Sharrim restate and reorganize, as
Aliyah's shaar does M20's structure.

**Display fonts are committed**, matching existing practice (Abbess and
Goudy have shipped in `chrome/fonts/` since the first folio); build.py
base64-embeds them and requires the files present. This is part of the
project's declared fan-work posture.
